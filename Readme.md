# Отчёт о горном походе — локальная сборка через Docker

Данный репозиторий содержит LaTeX-шаблон для отчёта о горном походе.
Сборка выполняется в изолированном Docker-контейнере, что гарантирует одинаковый результат на любой ОС.

## 📋 Требования

- **Docker** (версия 20.10 или новее)  
  Установка: [docker.com/get-started](https://www.docker.com/get-started)
- Docker должен быть **запущен** перед выполнением скриптов.

### Проверка Docker
- **Linux/macOS:** выполните `docker info` в терминале.
- **Windows:** откройте PowerShell и выполните `docker info`.
Если команда не найдена или выдаёт ошибку, Docker не установлен/не запущен.

## 🚀 Локальная сборка

### Linux / macOS

1. Откройте терминал в корневой папке проекта.
2. Дайте права на выполнение скрипта (один раз):
```bash
chmod +x build.sh
```

## 🔄 Частая пересборка документа (режим редактирования)

При активной работе над отчётом вы можете использовать **режим наблюдения** (`--watch` для Linux/macOS или `-watch` для Windows). В этом режиме контейнер остаётся запущенным, а `latexmk` автоматически перекомпилирует документ каждый раз, когда вы сохраняете изменения в `.tex`-файлах.

- **Linux/macOS:** `./build.sh --watch`
- **Windows PowerShell:** `.\build.ps1 -watch`

Для остановки нажмите `Ctrl+C`. Контейнер будет удалён автоматически (флаг `--rm`), а все временные файлы останутся на хосте.
